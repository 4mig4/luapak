luah_version (filename)
Reads version number from the given lua.h file.

Parameters:

  • filename string Path of the lua.h file.

Returns:

    string Version number in format x.y.

Or

  1. nil
  2. string An error message.
lualib_version (filename)
Reads version number of the given Lua library.

This function uses command strings to determine the version number. It works for both static and dynamic library on Linux and macOS.

Parameters:

  • filename string Path of the Lua library.

Returns:

    string Version number in format x.y.

Or

  1. nil
  2. string An error message.
find_incdir (lua_ver)
Looking for a directory containing lua.h in common locations.

Parameters:

  • lua_ver string Version of the lua.h file to search for.

Returns:

  1. string File path of the found directory.
  2. string Version of the found lua.h file in format x.y.

Or

    nil Not found.
find_liblua (lib_ext, lua_ver)
Looking for Lua library in common locations.

Parameters:

  • lib_ext string File extension of the library to search for.
  • lua_ver optional string Version of the Lua library to search for, or nil to search for any version.

Returns:

  1. string File path of the found Lua library.
  2. string Version of the found Lua library in format x.y.

Or

    nil Not found.
