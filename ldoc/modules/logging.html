Module logging

The logging module.

Fields:

  • output: (file) Where to write log messages (default: io.stderr).
  • prefix: (string) The prefix to add before every message (default: "luapak: ").
  • threshold: (int) The minimal log level that should be logged (default: 20).
  • DEBUG (10)
  • INFO (20)
  • WARN (30)
  • ERROR (40)
