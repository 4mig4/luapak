Functions

generate (lua_main, modules)

Generates source code of the C "wrapper" with the given Lua script and modules.

Module table:

  • type: "lua", or "native"
  • name: Full name of the module in dot-notation.
  • content: Source code or bytecode of Lua module (only for type "lua").

Parameters:

  • lua_main string Source code or byte code of the main Lua script.
  • modules {table,...} A list of modules to be included.

Returns:

    string A source code in C.

Raises:

if some module table doesn't have required keys or has wrong "type".
generate_from_files (main_file, modules)

Generates source code of the C "wrapper" with the given Lua script and modules.

Module table:

  • type: "lua", or "native"
  • name: Full name of the module in dot-notation.
  • path: Path of the Lua module to read (required for type "lua" if content is not set).
  • content: Source code or bytecode of the Lua module (required for type "lua" if path is not set).

Parameters:

  • main_file string Path of the main Lua script.
  • modules table A list of modules to be included.

Returns:

    string A source code in C.

Raises:

if any path in the modules is unreadable or some module table doesn't have required keys or has wrong "type".
